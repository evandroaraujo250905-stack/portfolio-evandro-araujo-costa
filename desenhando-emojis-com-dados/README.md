# 🎨 Sistema de Manipulação de Imagens e Matrizes Musicais

## 📝 Descrição do Projeto
Este projeto reúne diferentes desafios práticos voltados à manipulação de dados estruturados, explorando conceitos fundamentais de programação como matrizes, dicionários e transformação de dados.

A aplicação trabalha com três frentes principais:
- Processamento de imagens em formato de matriz (simulação de pixels)
- Transformação de estruturas musicais (transposição de matrizes)
- Integração e manipulação de dados visuais em listas e dicionários

O foco do projeto é desenvolver a lógica de transformação e organização de dados, simulando cenários reais de processamento gráfico e estrutural.

---

## 🖼️ Módulo 1: Processamento de Emoji (Imagem Matricial)
Este módulo simula um emoji utilizando uma matriz de cores RGB.

### 🔧 Funcionalidades:
- Representação de imagem como matriz de pixels
- Alteração de intensidade de cor (escurecimento)
- Renderização visual utilizando biblioteca gráfica

### 🎯 Objetivo:
Demonstrar como imagens podem ser manipuladas através de estruturas matriciais.

---

## 🎵 Módulo 2: Matrizes Musicais (Transposição)
Este módulo trabalha com dados musicais representados em matrizes.

### 🔧 Funcionalidades:
- Organização de frequências sonoras em estruturas bidimensionais
- Transposição de matriz (linhas → colunas)
- Reestruturação de dados musicais

### 🎯 Objetivo:
Simular transformação de dados musicais utilizando lógica de matrizes.

---

## 🔄 Módulo 3: Integração e Transformação de Pixels
Este módulo realiza modificações em uma coleção de imagens simplificadas.

### 🔧 Funcionalidades:
- Remoção de elementos da estrutura original
- Inversão de cores (efeito negativo)
- Inserção de novos elementos na estrutura

### 🎯 Objetivo:
Demonstrar manipulação dinâmica de listas e transformação visual de dados.

---

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python 3
* **Bibliotecas:** Matplotlib
* **Conceitos Aplicados:** Matrizes, Dicionários, Listas, Manipulação de Dados

O projeto permitiu consolidar conhecimentos importantes:

* **Manipulação de Matrizes:** Aplicação prática em imagens e dados musicais
* **Transformação de Dados:** Alteração estrutural de listas e dicionários
* **Visualização Gráfica:** Uso de biblioteca para exibição de dados visuais

## 🔧 Como Executar
1. Instale as dependências necessárias:
   `pip install matplotlib`

2. Execute o script Python:
   `python main.py`

3. Visualize os resultados no terminal e na janela gráfica gerada.
